# Event Hub

Event Hub is een Vlaamse WordPress-plugin voor infosessies en opleidingen. Je krijgt een eigen CPT voor events, een custom inschrijvingstabel, Elementor-widgets, geautomatiseerde e-mails en een moderne beheerervaring.

## Features
- Custom post type voor events met intuïtieve metabox UI
- Registraties in een aparte DB-tabel + exports en statusbeheer
- Automatische e-mails met sjablonen per eventfase
- Elementor-widgets (lijst + detail+formulier)
- Adminkalender met drag-to-create shortcut
- ReCAPTCHA en dubbele inschrijfcontrole

## Systeemeisen
- WordPress 5.8+
- PHP 7.4+
- Elementor (voor de widgets)

## Installatie
1. Plaats de map event_hub/ in wp-content/plugins/.
2. Activeer "Event Hub" in het WP-beheer.
3. Ga naar **Event Hub → Algemene instellingen** om de CPT-slug/reCAPTCHA/placeholder-opties te zetten.
4. Maak onder **Event Hub → E-mailsjablonen** je basis-sjablonen; koppel ze per event.
5. Voeg de Elementor-widgets toe aan de gewenste pagina's.
